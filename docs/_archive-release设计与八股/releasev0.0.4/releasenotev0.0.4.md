# Comet（彗记）v0.0.4 发布说明 —— 行者

**发布日期：** 2026年6月21日 ｜ **版本代号：** 行者（XingZhe · The Agent that Acts）

Comet v0.0.4 在 v0.0.3「群英」把助手拓展成「多角色协作平台」之后，迈出关键一步：让 Comet 从「会聊天的助手」升级为「**会自己干活的智能 Agent**」。这一版的主线是一条完整的自动化闭环——**一句话需求 → 自主深度研究（规划、多源检索、提炼、撰写）→ 结构化带来源的报告 → 可定时自动跑 → 结果主动送到你眼前甚至手机**。围绕这条主线，我们重构了研究引擎的质量（对标业界第一梯队的 GPT Researcher / STORM），补齐了「分享 / 导出 / 提醒 / 重试 / 推送」的交付闭环，并顺手把交互打磨得更像「人」——可拟人化的真人对话模式、可探索的知识图谱、可预览的知识库文档。当 Comet 能自己规划、自己查、自己写、到点自己跑、跑完主动告诉你，它就不再只是「等你提问的助手」，而是「替你奔走的行者」。

---

## 🚀 一、核心升级概览

### 1. 深度研究 → 报告（核心）🔬

- **一句话变一份报告**：给一个研究主题，Comet 自主完成「规划提纲与多视角子问题 → 四源检索（多查询联网搜索+并发抓正文 / 知识库 / MCP 增强）→ 分章节流式撰写并标注引用 → 汇总 TL;DR 与核心要点」，产出一份结构化、带可点来源链接的 Markdown 报告。全程实时活动流展示「正在搜什么、读了哪篇、写到哪节」，进度面板固定可见、完成自动收起，手机端历史抽屉+顶部工具条适配。
- **断线不怕**：研究在后台跑、事件经 Redis 通道续传，切页/关标签/断网都不影响它跑完落库，重连自动补推进度并续接。
- **质量重构（Deep Research v2）**：对标 GPT Researcher（CMU DeepResearchGym benchmark 第一）与斯坦福 STORM，定位 v1 质量瓶颈为「架构缺环」而非模型问题，重构为 8 步流水线——规划（多视角子问题）→ 检索 → **逐源提炼**（每源先 LLM 提炼成带来源号的「要点」，引用对齐提前、噪声当场过滤）→ **反思补搜**（找信息缺口补一轮，可关）→ **大纲整理**（要点编排成每节核心论点 + 证据分配）→ 分节写作（吃分配的要点 + 前文摘要避免重复）→ 汇总 → 拼装引用。
- **来源更可靠**：联网来源抓取后按「正文充实度 + 域名权威性」打分排序、丢弃抓取失败/登录墙的低质源，让有限的提炼预算落在可信资料上。
- **报告好用**：默认归档进专门的「深度研究报告」知识库（不污染其他库），可看 / 下载 Markdown / **导出 Word** / **打印为 PDF** / **生成公开分享链接** / 一键存入任意知识库。

---

### 2. 定时 / 主动任务 ⏰

- **到点自己跑**：建一个任务——一句话研究指令 + 触发规则（每天 HH:MM / 每周几 / 每隔 N 小时），到点系统自动跑一遍深度研究并落库报告。覆盖「每天追踪 AI Agent 秋招岗位」「每周一总结开源项目热点」等场景。
- **对话里随口建任务**：在对话中说「每天 9 点帮我查 X」，强模型会自动调用内置工具把任务建好并回复首次运行时间。
- **任务中心**：任务卡片 + 在线启停 + 立即运行一次 + 触发方式设置 + **运行历史**（每个任务跑过的报告，失败可见错误、一键重试）。
- **研究指令一键润色**：把口语化、笼统的研究指令润色成清晰聚焦、可执行的研究主题（深度研究主题框与定时任务指令框共用）。

---

### 3. 结果送达闭环 📬

- **首页 Agent 简报**：仪表盘新增「🤖 今日 Agent 简报」，汇总最近完成的研究报告（手动 / 定时区分图标），点击直达。
- **未读红点**：有新生成的报告时，侧边栏「定时任务」亮起未读角标，进页自动清零——结果主动提示，不用你去翻。
- **推到手机**：定时任务跑完，把报告 TL;DR + 链接主动推到你的手机/微信。支持 **Server酱 / 企业微信群机器人 / 钉钉群机器人 / 通用 Webhook** 四种渠道，每个用户配自己的密钥（Fernet 加密入库、只回显掩码），各推各的；可测试推送、按任务单独开关。不依赖邮件 SMTP。

---

### 4. 真人对话模式 🧑‍💬

- **一个开关，AI 像真人**：在「角色配置」开启全局真人模式后，所有单聊 / 群聊的 AI 回复都切换成真人微信聊天的样子——口语、简短、自然、有情绪，会开玩笑、会反问，用了工具也用大白话口语小结，绝不甩 markdown 标题/表格/免责声明那种「报告腔」。关闭即恢复助手风格。
- **多气泡逐条连发**：一条回复会像真人那样拆成几条短消息，配合「正在输入…」的停顿逐条蹦出来（停顿时长按句子长短模拟打字节奏），更有微信群聊的真实感。真人模式下还会隐藏「正在理解问题…」过程条与气泡下方的操作按钮，界面更纯粹。
- **设计取舍**：纯提示词驱动、不微调模型（对标 Claude 的 role prompting + 禁 AI 腔 + few-shot 思路）；多气泡用「模型输出分隔符、前端渲染拆分」实现，不改动流式协议，历史与分享一致。

---

### 5. 知识库与图谱 📚

- **文档内容预览**：知识库里的文档终于可以点开看内容了——Markdown 渲染、其余类型纯文本展示、网页来源附「查看原网页」，不用再靠文件名猜这篇是什么。
- **知识图谱重做（可探索）**：记忆图谱从「一次性平铺全部节点的毛球」改为「从焦点出发、点哪展开哪」的探索式图谱（react-force-graph）——拖拽顺滑、滚轮缩放、点节点展开其关联、搜索定位实体、hover 高亮邻居、按类型筛选、节点大小反映重要度。

---

### 6. 体验与工程打磨 🔧

- **真人 + AI 实时群聊**：多个真人与多个 AI 角色同群实时聊天（SSE 长连接 + Redis Pub/Sub），加入码邀请、在线状态、AI 用群主配置零配置加入。
- **单聊后台生成 + 断线续传**：单聊也做到生成与连接解耦，切页/断网 AI 照样把话说完并完整落库，重连续接。
- **重进对话默认打开上次会话**：进对话页直接接着上次聊，而不是每次都新开空白会话。
- **HTTPS 上线 + 合规**：Docker 化 nginx TLS 终止、域名证书、登录页 ICP 备案页脚。
- **每日回顾首屏异步化**：秒开旧内容 + 后台重生成，首屏不再卡几十秒。
- **MCP 工具提速**：编排器同轮结果缓存 + 持久 MCP 会话，整轮复用一次握手。
- **调度健壮性加固**：定时任务的研究执行拆到独立队列、加 Redis 锁防重叠跑、整体硬超时防卡死、原子认领防重复触发，调度心跳不被重活堵住。

## 🧭 未来展望

Comet v0.0.4 让产品第一次拥有了「自主完成一项工作」的能力：从规划、检索、提炼到成文，从手动发起到定时自跑，从首页简报到推送手机——一条「主动 Agent」的闭环跑通了。而 Deep Research v2 对标业界第一梯队的重构，也把「自动写出的报告」从「能看」推向「可信、有据、有结构」。

下一步，这位「行者」还能走得更远：深度研究可以引入「逐句溯源校验」「多 Agent 编辑—审阅循环」进一步提升可信度；定时任务可以做「增量对比」，让监控类任务只告诉你「相比上次有什么新变化」；主动关心可以延伸为基于长期画像与近期成果的主动建议。当 Comet 不仅替你查、替你写、还能替你盯着、替你判断什么值得你知道，它才真正配得上「行者」二字——替你奔走，也替你思考。

> 本版后端继续按功能补齐「设计与八股」归档（`docs/releasev0.0.4设计与八股/`），逐篇沉淀深度研究引擎、定时任务调度、报告分享导出、消息推送、真人对话模式等子系统的后端设计、数据模型、核心实现与面试问答。
