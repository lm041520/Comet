## V0.0.4 版本主题

让 Comet 从「会聊天的助手」升级为「**会自己干活的智能 Agent**」：一句话需求 → 自主深度研究 → 结构化带来源报告 → 可定时自动跑 → 结果主动送达（首页/手机）。围绕这条主线重构研究引擎质量（对标 GPT Researcher / STORM），补齐分享/导出/提醒/重试/推送闭环，并把交互打磨得更像人（真人对话模式、可探索知识图谱、文档预览）。明确不做代码沙箱。
